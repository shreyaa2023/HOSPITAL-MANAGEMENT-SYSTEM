# HOSPITAL-MANAGEMENT-SYSTEM
ER diagram and MySQL table conversion using XAMPP

## Description
This project contains an ER diagram and MySQL table conversion
for a Hospital Management System.

## Contents
- ER Diagram (PDF)
- MySQL SQL file for XAMPP (phpMyAdmin)

## Tools Used
- MySQL
- XAMPP
- phpMyAdmin

## How to Run
1. Open XAMPP and start Apache & MySQL
2. Open phpMyAdmin
3. Create database: hospital_management
4. Import the SQL file from Database folder
